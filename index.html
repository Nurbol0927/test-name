JavaScript Учебник

Типы данных

В JavaScript существует несколько основных типов данных:

  • Числа (Numbers): целые числа и числа с плавающей точкой.
  • Строки (Strings): последовательности символов, заключенные в кавычки.
  • Логический (Boolean): логическое значение true или false.
  • Undefined: значение переменной, если оно не было присвоено.
  • Null: значение, которое отсутствует или не существует.
  • Объекты (Objects): коллекции ключ-значение и объекты, включая функции.
  • Массивы (Arrays): упорядоченные списки значений.

alert, prompt, confirm

Встроенные функции для взаимодействия с пользователем:

  • alert: выводит диалоговое окно с сообщением.
  • prompt: выводит диалоговое окно с текстовым полем для ввода.
  • confirm: выводит диалоговое окно с подтверждением (OK/Отмена).

Пример использования:

alert("Привет, мир!");
let name = prompt("Введите ваше имя:", "");
let result = confirm("Вы уверены?");

Переменные

Переменная в JavaScript используется для хранения данных:

let age = 30;
const name = "John";

console.log

Метод console.log используется для вывода информации в консоль:

console.log("Привет, мир!");

if/else

Условные операторы if и else используются для выполнения кода при определенных условиях:

let age = 18;
if (age >= 18) {
  console.log("Доступ разрешен");
} else {
  console.log("Доступ запрещен");
}

?? и ||

Операторы ?? и || используются для работы с значениями по умолчанию:

let username = currentUser ?? "Гость";
let option = someOption || defaultValue;

for/while

Циклы for и while используются для повторения выполнения кода:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Функции

Функции в JavaScript используются для организации кода в более мелкие блоки:

function greet(name) {
  console.log("Привет, " + name);
}
greet("John");

switch

Оператор switch используется для выбора одного из множества блоков кода для выполнения:

let day = new Date().getDay();
switch (day) {
  case 0:
    console.log("Воскресенье");
    break;
  case 1:
    console.log("Понедельник");
    break;
  default:
    console.log("День недели");
}

Комментарии

Комментарии используются для описания кода и не выполняются интерпретатором JavaScript:

// Однострочный комментарий
/*
  Многострочный
  комментарий
*/

debugger

Команда debugger используется для остановки выполнения JavaScript кода в отладчике браузера:

function calc() {
  let x = 10;
  debugger;
  console.log(x * 2);
}
calc();

Object

Объекты в JavaScript используются для хранения коллекций данных и свойств:

let person = {
  name: "John",
  age: 30,
  greet: function() {
    console.log("Привет!");
  }
};
console.log(person.name); // "John"
person.greet(); // "Привет!"

this

Ключевое слово this ссылается на текущий объект в контексте выполнения кода:

let person = {
  name: "John",
  greet: function() {
    console.log("Привет, " + this.name);
  }
};
person.greet(); // "Привет, John"

Object.keys, values, entries

Методы Object.keys, Object.values и Object.entries используются для работы с объектами:

let user = {
  name: "John",
  age: 30
};
console.log(Object.keys(user)); // ["name", "age"]
console.log(Object.values(user)); // ["John", 30]
console.log(Object.entries(user)); // [["name", "John"], ["age", 30]]

new Date()

Конструктор new Date() создает объект Date, представляющий текущую дату и время:

let today = new Date();
console.log(today); // текущая дата и время

Рекурсия

Рекурсия в JavaScript - это процесс вызова функции из самой себя:

function countDown(num) {
  if (num <= 0) return;
  console.log(num);
  countDown(num - 1);
}
countDown(5);

Замыкание

Замыкание - это функция, которая запоминает свое лексическое окружение:

function makeCounter() {
  let count = 0;
  return function() {
    return count++;
  };
}
let counter = makeCounter();
console.log(counter()); // 0
console.log(counter()); // 1

setTimeout, setInterval

Функции setTimeout и setInterval используются для выполнения кода через определенный промежуток времени:

setTimeout(function() {
  console.log("Привет, мир!");
}, 2000);

let intervalId = setInterval(function() {
  console.log("Интервал");
}, 1000);

Остаточные параметры и оператор расширения

Остаточные параметры и оператор расширения позволяют работать с неопределенным числом аргументов:

function sum(...args) {
  return args.reduce((acc, val) => acc + val, 0);
}
let numbers = [1, 2, 3, 4];
console.log(sum(...numbers)); // 10

__proto__

Свойство __proto__ используется для доступа к прототипу объекта:

let animal = {
  eats: true
};
let rabbit = {
  jumps: true
};
rabbit.__proto__ = animal;
console.log(rabbit.eats); // true

Массивы

Массивы в JavaScript используются для хранения упорядоченных коллекций данных:

let numbers = [1, 2, 3, 4];
console.log(numbers.length); // 4
console.log(numbers[0]); // 1

BOM и DOM

Объекты BOM (Browser Object Model) и DOM (Document Object Model) предоставляют методы и интерфейсы для взаимодействия с браузером и HTML-документом:

// Пример работы с DOM
let element = document.getElementById("myElement");
element.textContent = "Новый текст";

fetch

Функция fetch используется для отправки HTTP запросов и получения ответов:

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Ошибка:', error));

cookie, localStorage

Куки и localStorage используются для хранения данных на стороне клиента:

// Установка значения в localStorage
localStorage.setItem('username', 'John');

// Получение значения из localStorage
let username = localStorage.getItem('username');

Регулярные выражения

Регулярные выражения используются для работы с текстовыми данными, для поиска и замены:

let text = "Пример текста";
let pattern = /текст/;
console.log(pattern.test(text)); // true

Промисы

Промисы используются для работы с асинхронными операциями и управления последовательностью выполнения кода:

let promise = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Готово!"), 1000);
});

promise.then(result => console.log(result)); // "Готово!"

Колбэки

Колбэки - это функции, которые передаются в качестве аргумента другой функции и вызываются после завершения операции:

function fetchData(callback) {
  setTimeout(() => {
    let data = "Данные загружены";
    callback(data);
  }, 2000);
}

fetchData(data => console.log(data)); // "Данные загружены"

async/await

Асинхронные функции и ключевые слова async/await используются для упрощения работы с асинхронным кодом:

async function fetchData() {
  let response = await fetch('https://api.example.com/data');
  let data = await response.json();
  return data;
}

Всплытие и погружение событий

Всплытие и погружение событий описывают порядок обработки событий в DOM:

document.addEventListener('click', function(event) {
  console.log('Кликнуто на элемент', event.target);
}, true);

JSON

JSON (JavaScript Object Notation) используется для передачи данных между сервером и клиентом:

let user = {
  name: "John",
  age: 30,
  isAdmin: false
};
let json = JSON.stringify(user);
console.log(json); // {"name":"John","age":30,"isAdmin":false}

setAttribute, getAttribute

Методы setAttribute и getAttribute используются для работы с атрибутами элементов DOM:

let element = document.getElementById('myElement');
element.setAttribute('data-id', '123');
let id = element.getAttribute('data-id');

Методы массивов

Методы массивов в JavaScript используются для манипуляции с элементами массива:

let numbers = [1, 2, 3, 4];
numbers.push(5); // добавление в конец массива
numbers.pop(); // удаление последнего элемента
numbers.splice(1, 1); // удаление элемента по индексу

Все операторы

В JavaScript существует множество операторов для выполнения различных операций:

// Арифметические операторы
let sum = 1 + 2;
let difference = 5 - 3;
let product = 4 * 6;
let quotient = 10 / 2;

// Операторы сравнения
let isEqual = 5 === 5;
let isNotEqual = 10 !== "10";
let greaterThan = 6 > 4;
let lessThanOrEqual = 3 <= 3;

// Логические операторы
let andOperator = true && false;
let orOperator = true || false;
let notOperator = !true;

// Операторы присваивания
let x = 5;
x += 2; // x = x + 2;

// Побитовые операторы
let a = 5 & 1; // побитовое И
let b = 5 | 1; // побитовое ИЛИ
let c = ~5; // побитовое НЕ
